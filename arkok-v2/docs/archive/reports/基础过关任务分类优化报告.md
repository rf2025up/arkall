## 📋 优化概要

根据用户反馈，对知识过关任务库中的基础过关分类进行了精细化调整，使分类更加符合实际教学需求，提升任务分类的准确性和实用性。

### 🎯 优化目标

1. **合并重复标签**: 将意思相同的任务类型合并
2. **优化标签命名**: 使用更准确的分类名称
3. **隐藏非基础任务**: 将不属于基础过关的任务设为隐藏
4. **增强分类逻辑**: 提升任务分类的教学针对性

---

## 🔧 具体优化内容

### 语文基础过关优化

#### ✅ 合并操作
- **古诗背诵 + 课文背诵** → **课文背诵**
- **古诗默写 + 课文默写** → **课文默写**

#### 📝 优化说明
- 古诗本质上是课文的一种形式，合并后分类更简洁
- 减少教师选择的困扰，提升操作效率
- 保持教学内容的完整性

### 数学基础过关优化

#### ✅ 标签优化
- **应用题** → **题型归纳**

#### 🔒 隐藏任务
- **几何图形和图形认知** → **图形认知**（隐藏，不属于基础过关）

#### 📝 优化说明
- "题型归纳"更能体现应用题训练的教学本质
- 图形认知不属于日常基础过关，设为隐藏避免干扰

### 英语基础过关优化

#### 🗑️ 删除任务
- **单词背诵**（删除）
- **句型背诵**（删除）

#### ✅ 保留任务
- **听力理解**（新增）
- **课文朗读**
- **单词默写**

#### 🔒 隐藏任务
- **口语对话**（隐藏，不属于基础过关）

#### 📝 优化说明
- 背诵类任务减少书面作业负担
- 保留默写类任务检测知识掌握程度
- 听力和朗读更适合基础过关场景
- 口语对话属于技能训练，非基础过关范畴

### 隐藏任务机制
- 通过 `hidden: true` 标记非基础过关任务
- 在数据插入时设置 `isActive: false`
- 前端界面不显示隐藏任务
