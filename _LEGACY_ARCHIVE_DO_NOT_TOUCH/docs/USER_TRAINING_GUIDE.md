## 🎯 系统概述

StarJourney是一个智能化的学习管理系统(LMS)，专为K-12教育设计，提供完整的备课、教学、学情管理解决方案。

### 核心功能

#### 📚 备课管理
- **任务库系统**: 11大类64个任务，覆盖所有学习场景
- **三科差异化进度**: 语文、数学、英语独立进度设置
- **智能QC配置**: 过关项个性化配置
- **个性化加餐**: 针对特定学生的额外任务

#### 📊 学情分析
- **实时数据监控**: 学生学习进度实时追踪
- **多维度分析**: 知识掌握、能力发展、成长轨迹
- **可视化展示**: 图表化数据呈现，直观易懂
- **学业地图**: 个性化学习路径规划

#### 🎮 游戏化激励
- **经验值系统**: 任务完成获得经验值
- **成就系统**: 勋章、等级、排行榜
- **社交互动**: PK对战、协作学习
- **即时反馈**: 学习成果即时展示

## 👥 用户角色

### 教师 👨‍🏫
**权限**: 备课发布、学情查看、个性化任务设置

**主要职责**:
- 制定和发布每日学习计划
- 配置基础过关(QC)和过程任务
- 设置个性化加餐任务
- 监控学生学习进度
- 调整教学策略

### 管理员 👩‍💼
**权限**: 全系统管理、用户管理、数据管理、系统配置

**主要职责**:
- 系统配置和维护
- 用户账户管理
- 数据备份和恢复
- 监控系统运行状态
- 生成教学报告

### 学生 👨‍🎓
**权限**: 查看个人任务、完成学习任务、查看学习成果

**主要职责**:
- 按时完成学习任务
- 参与互动活动
- 查看个人成长数据
- 与同学协作学习

## 👨‍🏫 教师操作指南

### 1. 系统登录

#### 访问地址
- **移动端**: `https://esboimzbkure.sealosbja.site/app`
- **管理后台**: `https://esboimzbkure.sealosbja.site/admin`

#### 登录步骤
1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 选择"教师"角色
4. 点击"登录"按钮

### 2. 备课发布流程

#### 📝 第一步：设置课程进度

**进入备课页面**:
- 点击底部导航"备课"
- 进入"今日备课"页面

**配置三科进度**:

**语文进度设置**:
```
单元: [3]   课文: [2]   标题: [古诗二首]
```

**数学进度设置**:
```
章节: [4]   节次: [1]   标题: [除法]
```

**英语进度设置**:
```
单元: [2]   标题: [Hello World]
```

#### ✅ 第二步：配置基础过关(QC)

**QC项目选择**:
- **语文QC**:
  - 生字听写 ✓
  - 课文背诵 ✓
  - 古诗默写
  - 课文理解

- **数学QC**:
  - 口算达标 ✓
  - 竖式计算
  - 公式背诵
  - 错题订正

- **英语QC**:
  - 单词默写 ✓
  - 句型背诵
  - 课文朗读

#### 📋 第三步：添加过程任务

**任务库使用**:
1. 点击"打开任务库"按钮
2. 浏览11大类专业任务：
   - 一、基础学习(核心) - 8个任务
   - 二、数学巩固 - 8个任务
   - 三、语文巩固 - 9个任务
   - 四、语文拓展 - 3个任务
   - 五、英语巩固 - 4个任务
   - 六、阅读任务 - 7个任务
   - 七、自主规划 - 4个任务
   - 八、整理与贡献 - 5个任务
   - 九、互助与创新 - 4个任务
   - 十、课堂成长 - 5个任务
   - 十一、家庭联结 - 7个任务

**推荐任务组合**:
```
• 完成数学书面作业 (+15 EXP)
• 课外阅读30分钟 (+15 EXP)
• 完成语文书面作业 (+15 EXP)
• 错题的红笔订正 (+12 EXP)
```

#### 🌟 第四步：设置个性化加餐

**目标学生选择**:
- 勾选需要特殊关注的学生
- 支持多选

**加餐任务配置**:
- 从历史任务中选择
- 或输入自定义任务
- 支持多任务组合

**示例**:
```
学生: 张小明, 李小花
任务: 背诵古诗 + 整理错题本 (+30 EXP)
```

#### 🚀 第五步：发布计划

**发布前检查**:
- ✅ 课程进度已设置
- ✅ QC项目已选择
- ✅ 过程任务已添加
- ✅ 个性化任务已配置

**点击"发布"按钮**:
- 系统自动为全班学生创建任务
- 支持Fan-out模式批量处理
- 实时反馈发布结果

**发布成功反馈**:
```
📊 计划ID: a9795fbb-c953-4e6d-8b92-91cb7e7e3669
📊 创建记录: 156个
📊 影响学生: 26名
📊 特殊任务: 8个
```

## ❓ 常见问题

### 教师常见问题

#### Q1: 如何修改已发布的备课计划？
**A**: 目前系统不支持修改已发布计划，建议：
- 发布新计划覆盖
- 联系管理员协助调整
- 在第二天重新发布

#### Q2: 为什么任务库显示空白？
**A**: 可能的原因和解决方案：
- 检查网络连接是否正常
- 刷新页面重新加载
- 联系技术支持

#### Q3: 个性化加餐任务不生效？
**A**: 检查步骤：
- 确认学生是否在名单中
- 验证任务是否正确设置
- 检查学生权限配置

#### Q4: 如何查看历史备课记录？
**A**: 功能开发中，临时解决方案：
- 记录备课日志
- 使用Excel表格管理
- 联系管理员查询数据库

### 学生常见问题

#### Q1: 任务完成后没有获得经验值？
**A**: 可能的原因：
- 任务需要老师审核确认
- 网络延迟导致数据同步慢
- 任务被标记为无效

**解决方法**:
- 耐心等待老师审核
- 刷新页面查看更新
- 联系老师确认

#### Q2: 忘记登录密码怎么办？
**A**: 重置密码方法：
- 联系班主任重置密码
- 请管理员协助重置
- 使用密码重置功能

#### Q3: 如何查看昨天的学习记录？
**A**: 查看方式：
- 进入"个人"页面
- 点击"学习历史"
- 选择日期查看记录

### 管理员常见问题

#### Q1: 如何批量导入学生数据？
**A**: 操作步骤：
1. 准备CSV格式数据文件
2. 使用"批量导入"功能
3. 映射字段对应关系
4. 验证导入结果

#### Q2: 系统运行缓慢怎么办？
**A**: 排查步骤：
1. 检查服务器资源使用情况
2. 分析数据库查询性能
3. 清理系统缓存
4. 联系技术支持

#### Q3: 如何处理数据备份异常？
**A**: 处理方法：
- 检查磁盘空间是否充足
- 验证数据库连接状态
- 查看备份服务日志
- 手动执行备份操作
